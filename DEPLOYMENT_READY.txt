================================================================================
                 REAL-TIME FEATURES IN ACTION
================================================================================

Admin Adds Product:
   1. Admin uploads on /admin/products
   2. Image uploaded to ImgBB
   3. URL saved to RTDB 'products' node
   4. Customer on /products page sees it INSTANTLY
   5. No page refresh needed!

Admin Changes Banner:
   1. Admin uploads new banner on /admin/settings
   2. URL saved to RTDB 'admin_settings/banner'
   3. Every visitor's home page updates INSTANTLY
   4. All changes live in real-time

Customer Places Order:
   1. Customer fills checkout form
   2. Order saved to RTDB 'orders' node
   3. Admin sees order INSTANTLY in /admin/orders
   4. No delay, no refresh needed!

Admin Updates Order Status:
   1. Admin changes status dropdown
   2. RTDB updated immediately
   3. Order list updates for admin
   4. Real-time sync across all pages

Admin Sets Delivery Charge:
   1. Admin enters amount on /admin/settings
   2. Saved to RTDB
   3. RTDB listener fetches it
   4. Checkout page shows INSTANTLY
   5. Next customer sees updated charge
