                    IF ADMIN BUTTON NOT SHOWING
========================================================================

DEBUG STEPS:

1. Open browser console (F12 or Ctrl+Shift+I)

2. Look for messages like:
   [v0] Auth context: {isAdmin: true, ...}
   [v0] User data from DB: {isAdmin: true, ...}

3. If isAdmin: false, check:
   - Email is EXACTLY: abdulhaseebkhatri123@gmail.com (case-sensitive)
   - No spaces before/after email
   - Firebase user was created

4. If Firebase errors:
   - Check .env.local has correct credentials
   - Firebase project exists
   - RTDB is created and initialized

5. Common issues:
   - Wrong email capitalization
   - .env.local not filled
   - Firebase project not created
   - Browser cache (clear and refresh)
