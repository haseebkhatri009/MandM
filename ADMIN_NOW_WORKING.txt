                    HOW IT WORKS NOW
========================================================================

FLOW:
1. User visits /admin
2. useEffect checks authLoading state
3. If still loading → show "Loading..." spinner
4. Once auth done, check user & isAdmin
5. If not admin → redirect to home (silently)
6. If admin → render admin dashboard

This ensures Firebase Realtime Database has time to:
- Fetch user data
- Check if email matches admin email
- Set isAdmin flag properly
